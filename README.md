Student ID Card Generator

📌 Project Overview

Student ID Card Generator is a GUI-based desktop application developed using Python and Tkinter. The application allows users to enter student details and generate a simple student ID card interface. This project was created to practice Python GUI development, event handling, input validation, image integration, and layout management using Tkinter.

---

✨ Features

- Enter student details through a graphical interface
- Input validation for empty fields
- Roll Number validation (digits only)
- Phone Number validation (digits only)
- Student photo display
- Dynamic output display after submission
- Success and error message boxes
- Clean and user-friendly interface
- Grid layout management

---

🛠️ Technologies Used

- Python 3
- Tkinter
- ttk Widgets
- MessageBox

---

📋 Student Information Fields

The application collects the following information:

- Name
- Roll Number
- Branch
- Year
- College Name
- Phone Number

---

🚀 How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Place the student image ("photo1.png") in the project folder.
4. Open the terminal in the project directory.
5. Run the following command:

python IDcard.py
